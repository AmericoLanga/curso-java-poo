# Curso de Java POO ☕

Repositório dedicado aos estudos de **Programação Orientada a Objetos em Java**, acompanhando as aulas do Gustavo Guanabara (Curso em Vídeo).

---

## 🚀 Projetos Implementados

### 📚 1. Sistema de Livraria e Leitura
Exercício focado na aplicação prática de **Encapsulamento** e **Interfaces**.
*   `Publicacao.java`: Interface que define os métodos abstratos de controle da leitura (abrir, fechar, folhear, avançar e voltar página).
*   `Livro.java`: Classe principal que implementa a interface `Publicacao`, contendo os detalhes do livro e o progresso da leitura.
*   `Pessoa.java`: Classe que representa o leitor, contendo atributos como nome, idade e sexo.
*   `objLivro.java`: Classe de execução (`main`) onde os objetos são instanciados e os métodos são testados.

### 🥊 2. Ultimate Emoji Combat (UEC)
Exercício focado no conceito de **Agregação entre Objetos** (Relacionamento "tem um").
*   `Lutador.java`: Classe que define os atributos de um lutador (nome, nacionalidade, idade, peso, categoria, vitórias, derrotas e empates) e métodos como status e treinar.
*   `Luta.java`: Classe que agrega os lutadores, responsável por marcar a luta (verificando se são da mesma categoria e se não são o mesmo lutador) e realizar o combate de forma aleatória.
*   `objLutador.java`: Classe de execução (`main`) responsável por criar o vetor de lutadores e simular os confrontos.
